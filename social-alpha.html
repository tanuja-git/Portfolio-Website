SOCIAL ALPHA

Revamping the look and experience of an interactive platform for entrepreneurs and innovators.

Challenge

Social Alpha is a platform for entrepreneurs and innovators to get feedback, funding, and advice. The site was already built from functionality point of view. With the increased number of users within the platform and some minor unlinked functions added over the period of time, came the necessity of revamping the UI and re-working minimal UX to better articulate the interactions happening within the platform.

Approach

When we started working, the product was feeling visually dull and clumsy. It was functioning, but it was not something you would be able to scan and understand easily.

We decided to work on the dashboard, as it is the main area of engagement for new and existing users and this is where all the major components come. To make the space useful, we added information about what is going on and segmented them to make it scan-able. By the end, we planned to create a UI kit that contains all components of the interface.

To make the dashboard useful as it should be, we added information about what is going on and segmented them to make it scan-able.
We revamped the entire UI of the platform and tweaked the UX a bit to make sense of the pages.

Essential actions placed upfront for efficiency

Social Alpha works with innovators, experts, and investors, they needed an interface that is time-efficient for every group. Engaging copy, clear CTA's, fewer buttons with icons, more labels with text, and all the essential actions on the problem/solution cards and its detail page placed thoughtfully were the key to achieve that.

We created modular cards for scalablity

Built for intuitive problem/solution creation

We made it easier for users to create problems/solutions by dividing it into multiple steps, the form need not be liner but capture all the necessary information before publishing. That’s what makes the process intuitive.

We divided adding problems/solutions are into multiple steps, four to be precise.
These four steps can be filled in any order.

Building a library of reusable components

We create a detailed UI library of components with atomic-inspired structure and naming conventions. We believe that with all UI elements in place, it will be easy for the Social Alpha team to add new functions internally.

For Social Alpha team to have flexibility of adding function quickly in the future, we shared a detail UI kit for their development team to work quickly and independently.
With the target users always on the go, the process is mostly done on a tablet or a mobile. We needed to make sure the essential information is still at hand – that's where the platform shines.
